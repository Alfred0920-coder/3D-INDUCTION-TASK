# 🎵 Spotify Music Trends — Data Analysis Project

This project explores a Spotify-style dataset of 1,500 tracks to uncover hidden patterns in music streaming behaviour using Python. 
It covers end-to-end data cleaning, exploratory data analysis (EDA), and advanced visualizations including a correlation heatmap and K-Means clustering to discover non-obvious trends — such as how acousticness negatively impacts popularity, and how Hip-Hop listening peaks late at night.
A Random Forest Regressor is also built as a bonus to predict track popularity from audio features like danceability, energy, and valence. 
All analysis is documented in a Jupyter Notebook with clear explanations alongside every chart.

Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
